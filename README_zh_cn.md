# NMSL-Launcher

[🇨🇳](README_zh_cn.md) [🇺🇸](README.md)

一个简单的 Minecraft 启动器。

> 目前版本暂不完善。

## 运行

```bash
./gradlew run
```

## 编译打包

```bash
./gradlew package
```
